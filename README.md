# 台灣密室逃脫知識庫 · Taiwan Escape Room Codex

> 把台灣密室逃脫的知識——遊戲題型、鎖具分類、發展史、產業數據、術語——一次整理成可互動、可搜尋的雙語靜態網站。

這是一個聚焦**台灣**的密室逃脫(實境解謎)知識整理站。內容涵蓋 16 種遊戲題型、16 類鎖具與觸發機關、主題類型、在台灣的發展簡史、產業概況數據、知名工作室地圖、新手攻略、術語速查,並附隨堂測驗與翻卡字卡。資料整理自公開的台灣與華語圈玩家指南、業者文章與百科,屬個人整理之非官方資源。

## ✨ 功能特色

- 🌏 **中英全頁切換** — 一鍵切換 zh / en,整站(卡片、詳情、導覽、搜尋框、標題)全部重繪,不留殘字
- 🌗 **深色 / 淺色模式** — 預設深色「密室」氛圍,可手動切換並記憶
- 🧭 **多頁面結構** — 11 個頁面各有獨立 URL 與 SEO,共用同一套導覽、語言與主題
- 🔍 **即時搜尋** — 題型、鎖具、主題、術語都能輸入關鍵字立即過濾
- 🏷️ **分類篩選** — 題型依「考驗的能力」、鎖具依「機構類型」用 chips 快速篩選
- 🔗 **深連結 + 詳情視窗** — 卡片點開 modal 詳解,網址帶 `#<slug>` 可直接分享
- 🎮 **每種題型附互動小遊戲** — 16 種題型各有一個可實際試玩的小遊戲(地毯式搜索、凱撒密碼、迷宮、UV 顯影、機關電路、磁鐵吸取、Simon 聲光…),點開題型卡片即可遊玩
- 🧩 **隨堂測驗** — 10 題即時 ✅/❌ 回饋與計分(只算當次)
- 🃏 **翻卡字卡** — 點擊翻面複習題型 / 鎖具 / 術語
- 📖 **術語速查** — 可搜尋的雙語詞彙表(含台灣玩家黑話)
- 🗺️ **互動地圖(可搜尋)、時間軸、儀表板** — 工作室地圖(Leaflet,含搜尋與北/中部篩選)、發展史時間軸、產業數據圖表
- 📱 **響應式設計** — 手機、平板、桌機皆適配
- ⚡ **純靜態零 build** — 無後端、無打包工具,載入快,可直接部署 GitHub Pages

## 🛠 本機使用

```bash
# 1. clone 專案
git clone git@github.com:tingwei161803/escape_room.git
cd escape_room

# 2. 啟動本機伺服器(建議,跨頁連結 / 深連結才正常)
uv run python -m http.server 4173
# 然後瀏覽 http://localhost:4173
```

> 本專案為純靜態網站,不需安裝任何依賴。依專案慣例,所有 Python 操作一律使用 `uv`(不要用裸 `python3` / `pip`)。

### 內容怎麼維護

所有內容都集中在 `data/data.js` 的 `window.SITE_PAGES` 陣列裡,每個 entry 對應一個頁面、自帶該頁版型所需的資料。要新增 / 修改題型、鎖具、術語等,直接編輯這個檔案即可,不需重新打包。
